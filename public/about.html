Welcome to Randevu Client

Randevu Client is a simple and efficient appointment booking system designed to make scheduling easy for both clients and service providers. Our localhost-based solution provides a clean, user-friendly interface for managing appointments.

Features

  • Easy appointment booking
  • View and manage your appointments
  • Multiple service types available
  • Responsive design for all devices
  • Local server hosting
  • Simple and intuitive interface

How It Works

Our system runs locally on your machine, serving HTML files through a simple Node.js server. This ensures:

  • Fast loading times
  • No internet dependency for basic functionality
  • Easy customization and development
  • Complete control over your data

Getting Started

To use the Randevu Client:

  1. Start the server by running npm start
  2. Open your browser and go to http://localhost:3000
  3. Navigate through the application using the menu
  4. Book appointments and manage your schedule

Technical Details

This application is built with:

  • Node.js for the server
  • Pure HTML, CSS, and JavaScript
  • No external dependencies
  • Mobile-responsive design
